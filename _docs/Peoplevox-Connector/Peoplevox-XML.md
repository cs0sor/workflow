To integrate with [Peoplevox](peoplevox) using Zynk you will need to import and export data using XML.  Imports require that templates are configured under Integration Templates, all fields required for the integration must be setup, and must use the default name.  We have created import files that will configure the templates as required.  Exports will return all fields documented under the related task.

## XML Formats

 * [Peoplevox Customer XML](peoplevox-customer-xml)
 * [Peoplevox Despatch XML](peoplevox-despatch-xml)
 * [Peoplevox Goods In XML](peoplevox-goods-in-xml)
 * [Peoplevox Item Availability XML](peoplevox-item-availability-xml)
 * [Peoplevox Item Movement XML](peoplevox-item-movement-xml)
 * [Peoplevox Item Type XML](peoplevox-item-type-xml)
 * [Peoplevox Purchase Order XML](peoplevox-purchase-order-xml)
 * [Peoplevox Sales Order XML](peoplevox-sales-order-xml)
 * [Peoplevox Suppliers XML](peoplevox-suppliers-xml)
