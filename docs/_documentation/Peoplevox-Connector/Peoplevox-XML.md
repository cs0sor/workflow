To integrate with [Peoplevox](peoplevox) using Zynk you will need to import and export data using XML.  Imports require that templates are configured under Integration Templates, all fields required for the integration must be setup, and must use the default name.  We have created import files that will configure the templates as required.  Exports will return all fields documented under the related task.
